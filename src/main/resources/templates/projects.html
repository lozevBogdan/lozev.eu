Sell8Buy

This project is my defense project for Java Web course at SoftUni (July 2022). It is an application where you can easily sell and buy anything, make registration, login, add or remove you product, send messages, make comments and management by admin.

Technologies: HTML, CSS, JavaScript JAVA, Spring Boot, Spring Data, Spring Security, Spring MVC, Hibernate, MySQL, Thymeleaf and REST.
